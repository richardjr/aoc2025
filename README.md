# aoc2025


## Setup

Add you sesson token
```bash
export AOC_SESSION=cafef00db01dfaceba5eba11deadbeef
```

Install dependencies
```bash
pip install -r requirements.txt
```

## Usage
Run a specific day's solution
```bash
python -m run --day 1
python -m run --day 1 --part b
python -m run --day 1 --live
python -m run --day 1 --live --submit
```
